# 경영학입문 팀 프로젝트

## 프로젝트 개요
이 프로젝트는 경영학입문의 팀 과제로, 창의적인 사업 아이템을 웹 페이지로 제작하여 발표하는 것을 목표로 합니다.

### 웹 페이지 링크
- [Netlify에서 프로젝트 보기](https://im-teamproject.netlify.app)

## 주요 기능
- 사업 아이템의 개념과 특징 설명
- 반응형 네비게이션 바 및 캐러셀 적용
- Bootstrap을 활용한 피처릿 섹션 제공
- 사용자 친화적인 디자인과 레이아웃

## 폴더 구조
